About YFinanceDash

This demo app showcases how to use the yfinance Python package to fetch market data, company info, and corporate actions. It's a prototype intended for research and learning.

Important: yfinance uses public Yahoo Finance endpoints. For production trading applications you should use a commercial data provider and implement stronger error handling, authentication and rate-limiting safeguards.
